### Learnings / patterns for future PRDs
- bun workspaces use an isolated linker: real packages live in root `node_modules/.bun/`,
  each workspace gets a `node_modules/` of symlinks. "Single hoisted node_modules" = this.
- Docs configs are fully self-contained (aliases `@/* -> ./*`, `collections/* -> ./.source/*`),
  so moving the dir as a unit needed zero path edits.
- `.source/` is generated by `fumadocs-mdx` postinstall; never commit/move it.
- Assistant does NOT run git (user owns all branches/commits/PRs). Next: PR 02 package scaffold.
